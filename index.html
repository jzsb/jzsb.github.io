How to install Nodeschool on Windows

  1. Download and install NodeJS
  2. Once done installing, open CMD. Windows + R and type cmd
  3. Now you want to check what version you have installed by typing "node -v" (without quotes). Make sure you have v5.1.0 or above
  4. Go to Nodeschool, scroll down until you see the Core section
  5. Copy the lenguage you want to install. For instance, I am going to install Javascripting so I will just copy "npm install -g javascripting"
  6. Paste the text you have copied into CMD and it will install. Once installed type the lenguage you have installed. Since I installed javascripting then I will just simply type "javascripting" into cmd and I will get blue screen with a list of challenges to do.
